Working With Directories And Files - Navigating The Terminal

Using the "cd" command

cd short for Change Directory

cd type "cd" any time and it will take you to your home directory

cd .. will bring you back up through your folder directory

cd /../.. will bring you 2 levels up your folder directory

cd - takes you back to the folder you were just in

cd / takes you to the lowest level file of your current directory

cp foldername copies directory to a new location

cp -r foldername Folder Junk/

cd d+tab will list all folders that start with the letter d (etc)

Managing Files and Folders

mkdir make directory / if spaces use quotation marks around the folder name

rm -d foldername deletes folder. You cannot delete a directory that has files within it

rm -rf foldername force removes the folder (handy if the folder has files within it)

rm filename deletes file

touch some_file.txt makes a new file

mv filename allows you to rename a file. Example: (mv another.rb file.rb). This command also allows you to move the file to another location. Example: (mv another.rb desktop/another.rb
