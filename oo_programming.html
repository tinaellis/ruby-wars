OOP

Classes

Naming CamelCase

Declaring box1 = Box.new

  • We declare objects of a class using new keyword. The above statement declares and object of the class Box.

Initialize def initialize

  • The initialize method is useful when you want to initialize some class variables at the time of object creation.

Class Method Defined def self.methodname()

  • A class method is defined using def self.methodname() which ends with end delimiter and would be called using class name as classname.methodname

Calling Class Methods classname.methodname

Class Variables @@running_total

  • Class variables store information per class hierarchy and follow the same rules as instance variables, except that they start with two at-signs. Class variables are shared between all instances of a class.

Class Constants CONSTANT

  • Once a constant is defined, you can not change its value but you can access a constant directly inside a class much like a variable but if you want to access a constant outside of the class then you would have to use classname::constant

Accessor Methods

  • def getWidth
  • @width
  • end

To make the variables available from outside the class, they must be defined within accessor methods, these accessor methods are also known as a getter methods. Object Oriented Ruby

Setter Methods

  • def setWidth=(value)
  • @width = value
  • end

Similar to accessor methods, which are used to access the value of the variables, Ruby provides a way to set the values of those variables from outside of the class using setter method. To use setter method: box.setWidth = 30

Access Control

Public Methods default

  • Public methods can be called by anyone. Methods are public by default except for initialize, which is always private.

Private Methods private

  • Private methods cannot be accessed, or even viewed from outside the class. Only the class methods can access private members.

Protected Methods protected

  • A protected method can be invoked only by objects of the defining class and its subclasses. Access is kept within the family.

Freezing Objects

Freeze Object.freeze

  • The freeze method in Object allows us to turn an object into a constant. A frozen object may not be modified: you can't change its instance variables.
